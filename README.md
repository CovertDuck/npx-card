# NPX Card

## About

My terminal business card.

## Usage

Fire up a terminal and run:

```sh
npx covertduck
```

## Building

Clone the repository:

```sh
git clone https://github/com/covertduck/npx-card.git
```

To install dependencies:

```bash
pnpm install
```

To run:

```bash
pnpm run build && pnpm run start
```
